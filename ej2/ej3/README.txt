Pinch to Zoom y Acelerómetro
Este proyecto permite interactuar con una figura en 
pantalla utilizando dos tecnologías: Pinch to Zoom para 
hacer zoom y rotación con dos dedos, y el Acelerómetro 
del dispositivo para ajustar el tamaño de la figura según
su inclinación. La figura cambia su tamaño y rotación 
según las interacciones del usuario.

Estructura del Proyecto

El proyecto está compuesto por los siguientes archivos:

index.html: Contiene la estructura básica de la página y carga los archivos CSS y JavaScript necesarios.
styles.css: Define el estilo para la página y la figura en el centro de la pantalla.
script.js: Contiene la lógica para capturar la inclinación del dispositivo y los gestos multitáctiles.


Funcionalidades

Acelerómetro:   La figura cambia de tamaño dependiendo de la inclinación 
                del dispositivo. Cuanto más inclinada esté la pantalla, 
                mayor será el zoom aplicado.
Multitáctil:    Permite hacer zoom y rotar la figura utilizando el gesto 
                de "pinch to zoom" con dos dedos.
